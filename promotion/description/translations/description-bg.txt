# Application:
OI Безопасност

# Promo text:
Съхранявайте личната си информация с OI Безопасност.

# Description:
OI Safe пази в безопастност всичките Ви пароли и други лични данни, използвайки AES шифроване на индустриално ниво. То също действа и като разширение за OI Notepad за да се шифроват бележки, създадени в OI Notepad. 

Това приложение е безплатно и не изисква интернет за ползването ѝ. 

За пълен списък с промените и често задаваните въпроси,моля посетете: 
http://www.openintents.org

Можете да помогнете за подобряване на превода на вашия език в Launchpad: 
https://translations.launchpad.net/openintents/trunk

Изходният код на това безплатно приложение с  отворен код е достъпен на адрес: 
http://code.google.com/p/openintents

Характеристики: 
* Предпазвай данни използвайки AES шифроване.
* Свободно местете елементи данни между категориите.
* Импортирайте/екстпортирайте елементи данни и бази данни.
* Направете резервни копия на/Възстановете елементи.
* Включва сигурен генератор на пароли.
* Автоматично прекъсване след пресрочване на период от време, за да се предпазват шифровани документи.
* Шифровайте OI Notepad бележки.
* Поддържа Obscura picture safe.
* MyBackup Pro.

Налични разширения за "OI File Manager": 
* Покажи информация за лиценза с "OI About".

Приложения, които работят със "OI File Manager": 
* Шифровайте създадени с OI Notepad бележки.
