# Application:
OI Safe

# Promo text:
Proteja sus datos privados con OI Safe.

# Description:
OI Safe protege todas sus contraseñas y otros datos privados usando el cifrado estándar AES. También actúa como una extensión de OI Notepad para encriptar notas. 

Esta aplicación no tiene anuncios y no requiere permisos de internet. 

Para la lista completa de cambios y la lista de preguntas frecuentes, por favor visite: 
http://www.openintents.org

Usted puede ayudar a mejorar la traducción de su idioma en Launchpad: 
https://translations.launchpad.net/openintents/trunk

El código fuente de esta aplicación libre y de código abierto está disponible en: 
http://code.google.com/p/openintents

Características: 
* Protege los datos usando el cifrado AES.
* Mueve libremente los registros de datos entre categorías.
* Importar/exportar registros de datos y bases de datos.
* Respaldar/restaurar registros.
* Incluye un generador seguro de contraseñas.
* Límite de tiempo automático para proteger documentos cifrados.
* Encriptar notas de OI Notepad
* Soporte para "Obscura picture safe".
* My Backup Pro.

Extensiones disponibles para "OI File Manager": 
* Mostrar la información de licencia en "OI About".

Aplicaciones que funcionan con "OI File Manager": 
* Encriptar notas creadas en "OI Notepad"
