# Application:
OI kluis

# Promo text:
Beveilig je privé-data met OI Kluis.

# Description:
OI Kluis beschermt privé-gegevens, zoals wachtwoorden, met de industriestandaard AES. Het is ook een extensie voor OI Notepad, om notities te versleutelen. 

Deze applicatie is advertentievrij, en heeft geen internet toegang nodig. 

Het complete overzicht van wijzigingen en de FAQ is te lezen op: 
http://www.openintents.org

Jij kan helpen om de vertaling in je eigen taal te verbeteren via Launchpad: 
https://translations.launchpad.net/openintents/trunk

De broncode van deze gratis en open source applicatie is beschikbaar op: 
http://code.google.com/p/openintents

Eigenschappen: 
* Data beschermen met AES versleuteling.
* Makkelijk data verschuiven van categorie naar categorie.
* Gegevens en databases importeren en exporteren.
* Gegevens back-uppen of terughalen.
* Inclusief veilige wachtwoordgenerator.
* Automatische time-out om beveiligde documenten te beschermen.
* OI Notepad-notities versleutelen.
* Ondersteuning voor Obscura picture safe.
* My Backup Pro

Beschikbare extensies voor "OI File Manager" 
* Toon licentie-informatie met OI About

Applicaties die werken met "OI File Manager" 
* Notities versleutelen die aangemaakt zijn met OI Notepad.
