# Application:
OI 금고

# Promo text:
Store your private data with OI Safe.

# Description:
OI 금고는 산업 표준 암호화 방식인 AES를 사용하여 당신의 비밀번호와 다른 비밀 정보를 보호합니다. 확장 기능으로 OI 메모장에서 만들어진 메모를 암호화하기 위해서도 쓰입니다. 

이 앱은 광고가 없으며 인터넷에 접속하지 않습니다. 

모든 변경 내역과 FAQ는 다음 주소에서 볼 수 있습니다: 
http://www.openintents.org

당신의 언어로의 번역을 Launchpad에서 도울 수 있습니다: 
https://translations.launchpad.net/openintents/trunk

이 무료 오픈 소스 앱의 소스 코드는 다음 주소에서 받을 수 있습니다: 
http://code.google.com/p/openintents

기능: 
* AES를 사용하여 정보를 안전하게 보호합니다.
* 데이터를 카테고리 간 자유롭게 옮기세요.
* 데이터베이스와 데이터 불러오기 및 내보내기.
* 데이터 백업/복원하기.
* 안전한 비밀번호 생성기를 포함합니다.
* 암호화된 문서를 보호하기 위한 자동 타임아웃.
* OI 메모장 메모 암호화 하기.
* Obscura 사진 금고에 대한 지원.
* MyBackup Pro.

사용 가능한 "OI 파일 관리자" 확장들: 
* "OI에 대해서" 로 라이선스 정보를 봅니다.

"OI 파일 관리자" 와 호환되는 앱들: 
* "OI 메모장"에서 작성된 메모 암호화하기.

키워드:
