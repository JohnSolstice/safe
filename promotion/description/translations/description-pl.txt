# Application:
OI Sejf

# Promo text:
Zapisuj swoje prywatne dane używając OI Sejf.

# Description:
OI Safe bezpiecznie chroni wszystkie hasła i inne prywatne dane przy użyciu standardowego szyfrowania AES. Działa również jako rozszerzenie do Ol Notatnik pozwalające na szyfrowanie notatek w nim utworzonych. 

Ta aplikacja jest darmowa i nie wymaga dostępu do internetu. 

Aby zobaczyć pełną listę zmian oraz FAQ, odwiedź: 
http://www.openintents.org

Możesz poprawić tłumaczenie na Twój język: 
https://translations.launchpad.net/openintents/trunk

Kod źródłowy tej darmowej i open-source'owej aplikacji jest dostępny pod adresem: 
http://code.google.com/p/openintents

Cechy: 
* Bezpieczna ochrona danych przy użyciu szyfrowania AES.
* Swobodnie przenosić dane pomiędzy kategoriami.
* Import / Eksport wpisów danych i baz danych.
* Kopia zapasowa Przywracanie wpisów.
* Włączając generatora bezpiecznych haseł.
* Automatyczny limit czasu chroniący zaszyfrowane dokumenty.
* Szyfrowanie notatek Ol Notatnik.
* Wsparcie dla bezpiecznego obrazu Obscura.
* MyBackup Pro.

Dostępne rozszerzenia dla "Ol File Manager": 
* Pokaż informacje licencyjne za pomocą "OI About".

Aplikacje działające z "OI Menedżer plików": 
* Szyfruj notatki utworzone w "Ol Notatnik".
