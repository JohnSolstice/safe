OI Safe droši aizsargā visas Jūsu paroles un citus privātos datus izmantojot nozares standartu AES šifrēšanu. Tas arī darbojas kā paplašinājums OI Notepad, lai šifrētu piezīmes, kuras ir izveidotas ar OI Notepad. 

Šī aplikācija ir bezmaksas un tai nevajag interneta atļauju. 

Lai skatītu pilnīgu izmaiņu sarakstu un sarakstu ar biežāk uzdotajiem jautājumiem, lūdzu apmeklējiet: 
http://www.openintents.org

Jūs varat palīdzēt uzlabot tulkojumu savā valodā izmantojot Launchpad: 
https://translations.launchpad.net/openintents/trunk

Pirmkods ir bezmaksas un atvērtā pirmkoda aplikācija ir pieejama: 
http://code.google.com/p/openintents

Funkcijas: 
* Aizsargājiet datus droši izmantojot AES šifrēšanu.
* Brīvi pārvietojiet datu ievades starp kategorijām
* Importējiet/eksportējiet datu ievades un datubāzes.
* Dublējiet/atjaunojiet ievades.
* Iekļauj drošu paroļu ģeneratoru.
* Automātisks noilgums, lai aizsargātu šifrētos dokumentus.
* Šifrējiet OI Notepad piezīmes.
* Atbalsts Obscura attēlu seifam.
* MyBackup Pro.

Pieejamie paplašinājumi "OI File Manager": 
* Parādīt licenzes informāciju ar "OI About".

Aplikācijas, kuras strādā ar "OI File Manager": 
* Encrypt notes created in "OI Notepad".
