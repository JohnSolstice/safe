# Application:
OI بىخەتەرلىك

# Promo text:
شەخسىي سانلىق مەلۇماتلىرىڭىزنى OI بىخەتەرلىك بىلەن ھەمبەھىرلەيدۇ.

# Description:
OI Safe safely protects all of your passwords and other private data using industry standard AES encryption. It also acts as an extension to OI Notepad to encrypt notes created in OI Notepad. 

بۇ ئەپتە ئېلان يوق، ئىنتېرنېت ھوقۇقى تەلەپ قىلمايدۇ. 

ئۆزگىرىشنىڭ تەپسىلىي تىزىمىنى تۆۋەندىكى ئادرېستىن كۆرۈڭ: 
http://www.openintents.org

بۇ ئەپنى Launchpad تە ئۆز تىلىڭىزغا تەرجىمە قىلالايسىز: 
https://translations.launchpad.net/openintents/trunk

كودى ئوچۇق بۇ ئەپنىڭ ئەسلى كودى: 
http://code.google.com/p/openintents

ئىقتىدارلار: 
* Securely protect data using AES encryption.
* سانلىق مەلۇمات تۈرلىرىنى كاتېگورىيەلەر ئارىسىدا ئەركىن يۆتكەيدۇ.
* سانلىق مەلۇمات تۈرى ۋە ساندانلارنى ئەكىر/چىقار.
* تۈرلەرنى زاپاسلا/ ئەسلىگە كەلتۈر.
* بىخەتەر ئىم ھاسىل قىلغۇچنى ئۆز ئىچىگە ئالىدۇ.
* Automatic timeout to protect encrypted documents.
* OI خاتىرە دەپتەر خاتىرىلىرىنى شىفىرلايدۇ.
* Support for Obscura picture safe.
* MyBackup Pro.

«OI ھۆججەت باشقۇرغۇچ» نىڭ ئىشلەتكىلى بولىدىغان كەڭەيتىلمىلىرى: 
* «OI ھەققىدە» ئىجازەت ئۇچۇرىنى كۆرسىتىدۇ.

«OI ھۆججەت باشقۇرغۇچ» تا ئىشلەيدىغان ئەپلەر: 
* «OI خاتىرە دەپتەر» دە قۇرۇلغان خاتىرىنى شىفىرلايدۇ.
